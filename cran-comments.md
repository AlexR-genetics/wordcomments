## Package Description

wordcomments provides tools for extracting, analyzing, and managing comments 
from Microsoft Word (.docx) documents. It is particularly useful for academic 
manuscript revision workflows where authors need to respond to reviewer comments.

Key features:
- Extract comments with metadata (author, date, position, resolution status)
- Detect comment threads (replies to comments)
- Generate response tables for journal revisions
- Export to multiple formats (Excel, Word, CSV, Markdown, HTML)
